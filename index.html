01

KTNB Travel — Booking Platform

In production

A full-stack travel platform for a Malaysian agency handling four booking journeys — flights, tour & Umrah packages, a product store and hotels — for guests and members, with a complete admin operations module behind it.

  • Sole engineer. Designed and built the entire platform — customer storefront, four booking flows, and the admin back office.
  • Integrated Sabre GDS for real-time flight search and e-ticket issuance, plus the Agoda hotel affiliate flow and commission tracking.
  • Engineered a race-safe inventory reservation system with TTL expiry that prevents overselling during concurrent bookings.
  • Built resilient payments on FPX / Billplz — HMAC-verified, idempotent webhooks, a 3-retry limit, and a polling fallback for stuck transactions.
  • Shipped the full refund lifecycle (request → admin approval → settlement) and an append-only, 7-year audit log.

MyKopTNB — Member Super App

30,000+ members
Admin Dashboard overview · 30,603 members
1 / 5

A cooperative super-app for 30,000+ members of KopTNB — the cooperative of TNB, Malaysia's national utility — pairing a React Native mobile app with a Next.js admin backend over one versioned API.

  • Built core mobile features: e-wallet (top-up, transfer, pay-by-QR), an in-app store, forms & surveys, and SMS-OTP member onboarding.
  • Shipped an on-tablet election voting system — kiosk voting at physical booths with publicly visible live results — across app and admin.
  • Designed a dual-database architecture: app data in PostgreSQL plus the read-only MSSQL member registry, with safe fallback when upstream is down.
  • Implemented triple-transport auth (mobile Bearer, admin cookie, external M2M) on a single Logto-backed resolver with local role authority.
  • Backend is a 65-table schema across 32 migrations serving mobile, admin and machine-to-machine callers from one code path.
